# IoT Temp Watch

**IoT Temp Watch** is a full-stack mini project that collects real-time temperature data from a public sensor API and displays it on a modern dashboard.

**Project Features:**
- Fetches temperature or humidity data from a public IoT API.
- Stores and serves data through a Python Flask backend.
- Displays data in real-time using a React frontend.
- Optional AI integration (LLM, RAG, deep learning models).
- Simple authentication, clean code, and Git best practices.

**Tech Stack:**
- Backend: Python, Flask, SQLite
- Frontend: React
- Optional: WebSocket, Docker, CI/CD

**API Endpoints:**
- `/data/latest` – Get the latest temperature with trends
- `/data/history` – Get historical temperature data
